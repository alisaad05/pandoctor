jmpSetColdfireCalculationType()

Defines the Coldfire calculation type.

Syntax

int jmpSetColdfireCalculationType (int typeFlag);

Parameters

typeFlag [input]

Integer that specifies the type of calculation to perform when running Coldfire. This parameter can take one of the following values:

JMP_COLDFIRE_CALCULATION_PHYSICAL_PROPERTIES To calculate physical and thermophysical properties.
JMP_COLDFIRE_CALCULATION_MECHANICAL_PROPERTIES To calculate room-temperature matrix mechanical properties, with contributions from solid solution and grain boundary strengthening.

Return values

JMP_STATUS_OK Request has been successfully completed.
JMP_STATUS_ERROR An error has occurred.

Remarks

  • This function call is optional. By default, Coldfire performs the calculation of physical and thermophysical properties.
  • Depending on the specified calculation type, different functions are required to complete the setup of the Coldfire module.
  • In Coldfire calculations, Poisson’s ratio, as well as Young’s, bulk, and shear moduli are classified as physical properties.
  • Currently, the calculation of room-temperature matrix mechanical properties is only possible for steels, aluminium, cobalt, and titanium alloys, as well as nickel and nickel-iron based superalloys.
