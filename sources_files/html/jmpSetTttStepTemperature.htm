jmpSetTttStepTemperature()

Defines the step temperature used for TTT calculations.

Syntax

int jmpSetTttStepTemperature (double stepTemperature);

Parameters

stepTemperature [input]

Double that specifies the temperature interval between the various isothermals considered in a TTT calculation.

Return values

JMP_STATUS_OK Request has been successfully completed.
JMP_STATUS_ERROR An error has occurred.

Remarks

  • This function call is optional. If you do not specify the TTT step temperature, a default value of 10 °C will be assumed.
  • The input temperature unit may be specified using jmpSetTemperatureUnit(). Note, however, that in this case changing between kelvins and degrees Celsius has no effect.
