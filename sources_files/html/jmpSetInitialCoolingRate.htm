jmpSetInitialCoolingRate()

Defines the initial cooling rate used for TTT calculations.

Syntax

int jmpSetInitialCoolingRate (double rate);

Parameters

rate [input]

Double that specifies the cooling rate (in °C/s) from the austenitisation temperature to the various isothermal holding temperatures considered in a TTT calculation.

Return values

JMP_STATUS_OK Request has been successfully completed.
JMP_STATUS_ERROR An error has occurred.

Remarks

  • This function call is optional. If you do not specify the initial cooling rate, a default value of 106 °C/s will be assumed.
  • In general, the specified rate should be sufficiently fast to prevent phase transformations during the cooling process.
