jmpSetAlloyElements()

Defines how many and which elements are present in the alloy.

Syntax

int jmpSetAlloyElements (int numElements, char **elementList);

Parameters

numElements [input]

Integer that specifies the number of elements in the alloy.

elementList [input]

Pointer to an array of null-terminated sequences of characters that specifies the list of elements in the alloy.

Return values

JMP_STATUS_OK Request has been successfully completed.
JMP_STATUS_ERROR An error has occurred.

Remarks

  • This function call is required to set up most calculation modules. The only exception is Coldfire, for which the elements in the alloy are read from a Solver output file.
  • Depending on the material type, only specific sets of elements are allowed. Valid choices are listed below:
    • Aluminium alloys Al, B, Bi, C, Ca, Co, Cr, Cu, Fe, H, La, Li, Mg, Mn, Mo, Ni, Pb, Sc, Si, Sn, Sr, Ti, V, Zn, Zr.
    Magnesium alloys Mg, Al, Ca, Ce, Cu, Fe, Gd, La, Mn, Nd, Sc, Si, Sn, Sr, Y, Zn, Zr.
    Zirconium alloys Zr, C, Cr, Fe, H, Hf, N, Nb, Ni, O, Si, Sn.
    Titanium alloys Ti, Al, B, C, Cr, Cu, Fe, H, Mo, N, Nb, Ni, O, Re, Ru, Si, Sn, Ta, V, Zr.
    Cobalt alloys Co, Al, B, C, Cr, Fe, Hf, Mn, Mo, N, Nb, Ni, Si, Ta, Ti, W, Zr.
    Solder alloys Sn, Ag, Al, Au, Bi, Cu, In, Ni, Pb, Sb, Zn.
    Nickel-based superalloys Ni, Al, B, C, Co, Cr, Cu, Fe, Hf, Mn, Mo, N, Nb, O, Pt, Re, Ru, Si, Ta, Ti, V, W, Zr.
    Nickel-iron-based superalloys Ni, Al, B, C, Co, Cr, Cu, Fe, Mn, Mo, N, Nb, Si, Ta, Ti, W.
    Nickel-based single crystals Ni, Al, B, C, Co, Cr, Hf, Mo, Nb, Pt, Re, Ru, Si, Ta, Ti, W.
    Cast irons Fe, Al, C, Cr, Cu, Mg, Mn, Mo, N, Nb, Ni, P, S, Si, Ta, Ti, V.
    Stainless steels Fe, Al, C, Co, Cr, Cu, Mn, Mo, N, Nb, Ni, O, P, S, Si, Ti, V, W.
    General steels Fe, Al, B, C, Co, Cr, Cu, Mn, Mo, N, Nb, Ni, O, P, S, Si, Ta, Ti, V, W.
    Copper alloys Cu, Al, Cr, Fe, Mg, Mn, Ni, P, Pb, Si, Sn, Zn.
  • The base element for a given material type (the first in the sets above, e.g., Fe for steels) should always come first in the specified list of elements.
