jmpSetAustenitisationTemperature()

Defines the austenitisation temperature for general steels.

Syntax

int jmpSetAustenitisationTemperature (double temperature);

Parameters

temperature [input]

Double that specifies the temperature used to austenitise a general steel.

Return values

JMP_STATUS_OK Request has been successfully completed.
JMP_STATUS_ERROR An error has occurred.

Remarks

  • This function is only relevant to set up the TTT and CCT modules.
  • This function call is optional. If you do not specify the austenitisation temperature, a default value of 860 °C will be assumed. Note, however, that 860 °C may be lower than the A3 temperature of the steel of interest.
  • The input temperature unit may be specified using jmpSetTemperatureUnit().
