Troubleshooting

If an error occurs while running an application that uses the JMatPro API, a message will usually be displayed explaining the problem. The following describes the most frequently encountered issues and how you may resolve them:

  • An error message indicates that a file cannot be found. To resolve this issue, you must ensure that your application can access the required DLLs, configuration and database files at all times. These are provided in the subdirectories Runtime/win32 and Runtime/x64 in the API installation directory, which should be used for 32-bit and 64-bit applications, respectively. The contents of the appropriate subdirectory should be placed at the same level as your executable file and its structure should be kept intact.
  • An error message indicates that a calculation is already running. Since the JMatPro API is not thread-safe, attempting to run several calculations in concurrent threads will result in failure. Simply wait for a calculation to finish before you begin setting up and running a new one.
  • An error message indicates that the Sentinel USB key cannot be found. Should this happen, you must first ensure that the Sentinel key is correctly plugged in. If the problem persists, you should check whether the Sentinel key is being recognised by your system. If you are connecting remotely, make sure that the runtime drivers are properly installed, and that a firewall or anti-virus software is not blocking the TCP/UDP port 1947. Also, you may be having trouble picking up the right key, in which case specifying the precise server name (or IP address) in the first line of the file configs/system.def may solve the issue. Otherwise, you may be experiencing a network problem and should seek the help of your network administrator.

    • Note  You may monitor the Sentinel keys available on your machine (that can be reached locally or through a remote server) by following the link http://localhost:1947/_int_/devices.html. Provided that the runtime drivers are correctly installed, the Sentinel Admin Control Center web page should appear with the desired information.

  • An error message indicates that the license is invalid. If you receive this or a similar message, and are not experiencing any issues with the Sentinel protection key, then your license may not be appropriate to use a specific API functionality, or it may have expired. If you wish to obtain a new license, or renew an existing one, please contact a Sente Software representative.

Calculation tips

Occasionally, you may come across a particularly difficult calculation that takes a long time to run or even fails because of numerical instabilities. To help mitigate these issues, you may observe the following suggestions:

  • To avoid unexpected or unreliable results, the alloy composition should be consistent with the material type.
  • To obtain a more robust calculation, consider removing impurity elements with a very small concentration.
  • For Solver stepping calculations, you may use a different starting point (especially a higher temperature) and/or reduce the step size.
  • For Scheil-Gulliver calculations, it is recommended to use a sufficiently small step size, of 5 °C or less.
  • Searching for phase boundaries is a difficult task which may cause problems in Solver calculations. If not necessary, consider disabling this feature, as doing so will generally lead to faster and less error-prone calculations.
